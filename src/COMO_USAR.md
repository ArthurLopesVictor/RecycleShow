# 🎮 Como Usar o Recycle Show

## 🚀 Início Rápido

### 1️⃣ Primeira Vez no App

1. **Abra o Recycle Show** no navegador
2. Você verá a tela de boas-vindas com duas opções:

#### Opção A: Criar Nova Família
```
→ Clique em "Criar Nova Família"
→ Digite o nome da sua família (ex: "Família Silva")
→ O sistema gera automaticamente um código de 6 caracteres (ex: "ABC123")
→ ANOTE ESTE CÓDIGO! Você vai precisar para adicionar outros membros
→ Crie seu primeiro jogador (nome + avatar)
→ Pronto! Você está dentro! 🎉
```

#### Opção B: Entrar em Família Existente
```
→ Clique em "Entrar em Família Existente"
→ Digite o código de 6 caracteres que recebeu
→ Selecione um jogador existente OU crie um novo
→ Pronto! Você está dentro! 🎉
```

---

## 👥 Gerenciando Membros da Família

### ✨ NOVIDADE: Adicionar Membro Sem Sair do Jogo!

1. **Clique no seu avatar** no canto superior direito
2. Um menu dropdown abre mostrando todos os membros
3. **Clique em "Adicionar Membro"** no final da lista
4. Um diálogo abre onde você pode:
   - Digitar o nome do novo membro
   - Escolher um avatar
   - Clicar em "Adicionar"
5. **Pronto!** O novo membro aparece na lista instantaneamente
6. **Sem necessidade de fazer logout ou voltar ao início!**

### Trocar de Jogador

1. **Clique no seu avatar** no header
2. Veja todos os membros da família
3. **Clique no membro** que você quer usar
4. O sistema carrega automaticamente os dados dele
5. Todos os jogos, pontos e estatísticas são do jogador selecionado

## 🎮 Jogando

   - Puzzle estratégico com mapa visual

## 💾 Exportar Dados

1. No **footer** da página, encontre "Exportar Dados"
2. Clique em "Baixar Métricas"
3. Escolha o formato:
   - **JSON** - Para análise programática
   - **CSV** - Para abrir no Excel/Planilhas
4. O arquivo é baixado automaticamente com todas as jogadas do jogador atual

## 🔐 Sessão e Logout

### Sessão Automática
- O sistema salva sua sessão automaticamente
- Quando você voltar, já estará logado
- Funciona até fechar o navegador

### Fazer Logout
1. Clique no botão **"Sair"** no canto superior direito
2. Você volta para a tela de login
3. Sua família e jogadores continuam salvos no banco de dados
4. Pode fazer login novamente a qualquer momento

## 🐛 Solução de Problemas

### "Token inválido"
**Problema:** O código da família não está funcionando
**Solução:**
- Verifique se digitou corretamente (6 caracteres)
- Certifique-se que o código é de letras maiúsculas e números
- Peça o código novamente para quem criou a família

### "Erro ao carregar sessão"
**Problema:** O app não carrega seus dados
**Solução:**
1. Abra as **Ferramentas do Desenvolvedor** (F12)
2. Vá em **Application** → **Local Storage**
3. Limpe os itens:
   - `recycle_token_familiar`
   - `recycle_current_player_id`
4. Recarregue a página e faça login novamente

### Família não aparece
**Problema:** Criou a família mas não consegue acessar
**Solução:**
- Verifique sua conexão com a internet
- Aguarde alguns segundos e tente novamente
- Se persistir, crie uma nova família

### Membros não aparecem
**Problema:** Adicionou um membro mas ele não aparece
**Solução:**
- Clique no avatar novamente para atualizar a lista
- Faça logout e login novamente
- O membro deve estar lá!
